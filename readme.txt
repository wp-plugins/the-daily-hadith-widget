A widget that displays a different hadith a day

== Description ==

A widget that displays a different hadith a day. It takes hadiths from 
Sahih Bukhari, Sahih Muslim and more. It also gives a link at the end 
of the hadith to iknowledge.islamicnature.com where you can read more hadiths.
Tags: koran, quran, muslim, islam, plugin, hadith, widget.

== Installation ==

1. Extract the folder into the wp-content/plugins directory.
2. Go to the plugins section in the admin backend and activate the widget.
3. Go to Appearance > Widgets and add the widget to your sidebar.
4. Enjoy.
